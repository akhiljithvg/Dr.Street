# duckie_perception

This package contains the main perception node for the Duckie robot.

## Active node

- `duckie_perception/duckie_perception/followlaneesp_node.py`

## Responsibilities

- camera input processing
- red lane detection
- ArUco junction detection
- serial motor command output
